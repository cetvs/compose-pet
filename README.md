Использовал:
- Coroutines
- Flow для передачи данных из Retrofit/Room
- MVVM (хранит state для передачи данных в UI) 
- Hilt для инъекции зависимостей
- Clean Architecture

UI взят из тестового, а также добавленна внутренняя логика.
